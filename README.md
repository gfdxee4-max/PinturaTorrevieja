# PinturaTorrevieja

Commercial website foundation for an automotive paint and body workshop, built with Next.js 15, TypeScript, Tailwind CSS, and the App Router.

## Scripts

```bash
npm run dev
npm run build
npm run typecheck
npm run lint
```

## Structure

- `src/app` - App Router routes, metadata, sitemap, robots.
- `src/components/layout` - Site-level shell components.
- `src/components/sections` - Page section building blocks.
- `src/components/ui` - Reusable UI primitives.
- `src/config` - SEO and site configuration.
- `src/lib` - Utility functions.
